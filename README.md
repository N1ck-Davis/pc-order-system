# PC Ordering System

This is a Java coursework project for the **Advanced Programming** module at **Newcastle University**.

The project demonstrates object-oriented programming concepts such as classes, interfaces, inheritance, and polymorphism.  
It simulates a system where customers can order preset or custom PCs and manage their orders.

## Features
- Create and manage custom or preset PC models  
- Handle customer and order management  
- Validate credit card information  
- Use of OOP principles  
- Includes JUnit tests  

## Technologies Used
- Java  
- JUnit  
- Maven  
- IntelliJ IDEA  

## How to Run
1. Clone this repository  
   ```
   git clone https://github.com/N1ck-Davis/pc-order-system.git
   ```
2. Open the project in **IntelliJ IDEA**  
3. Build the project using **Maven**  
4. Run the main class or test files
